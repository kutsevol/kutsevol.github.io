Моя первая статья используя Pelican!

Pelican - это генератор статических сайтов, который написан на Python.

Генератор статических сайтов преобразовывает тест в статические файлы HTML, CSS и IMAGE. После данного преобразования остается лишь загрузить данные файлы на WEB-сервер, и сайт готов.
